Tablas

Las tablas se utilizan para tabular datos, no para diseñar páginas web.

Etiquetas importantes:

  • table (Para crear una tabla)
  • tr (Para crear una fila) Table Row
  • th (Para crear la celda de cabecera) Table Header Por defecto aparecen en negrita y centradas
  • td (Para crear una celda de fila) Table Data
  • caption (Para poner título a una tabla)
    • Etiquetas añadidas en HTML5
  • thead (Agrupación de cabecera)
  • tbody (Cuerpo de la tabla)
  • tfood (Agrupación del pie)
